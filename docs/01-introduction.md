# 1. Introdução

## 1.1 Visão Geral

O Finanpy é um sistema de gestão de finanças pessoais desenvolvido com Python e Django, focado em simplicidade e eficiência. O projeto adota uma abordagem full-stack utilizando Django Template Language com TailwindCSS para o frontend, proporcionando uma experiência moderna e responsiva.

A aplicação permite aos usuários gerenciar suas finanças de forma intuitiva, oferecendo controle de contas bancárias, categorização de transações, visualização de entradas e saídas, e um dashboard centralizado para análise financeira.

## 1.2 Propósito

Fornecer uma ferramenta simples, eficiente e acessível para que pessoas possam organizar suas finanças pessoais, acompanhar receitas e despesas, categorizar transações e tomar decisões financeiras mais conscientes através de visualizações claras e objetivas.

## 1.3 Público Alvo

- **Perfil Primário**: Adultos entre 25-45 anos que buscam organizar suas finanças.
- **Perfil Secundário**: Jovens profissionais iniciando sua independência financeira.
- **Características Comuns**:
  - Possuem múltiplas contas bancárias.
  - Necessitam controlar gastos mensais.
  - Buscam simplicidade e praticidade.

## 1.4 Objetivos

- **Produto**: Permitir cadastro seguro, gerenciamento de contas, registro de transações e visualização clara do panorama financeiro em um dashboard.
- **Usuário**: Ter uma visão consolidada de todas as contas, entender para onde o dinheiro está indo, controlar entradas e saídas e acessar as informações de forma rápida.
